# adventOfCode2018
Mes solutions pour advent of code 2018 (golang pur)

Le fichier d'entrée doit être placé au même endroit que l'exécutable et nommé `input`

## Exemple

```
$ cd day1
$ ls
day1.go  input
$ go build
$ time ./day1
first star: 493
second star: 413

real    0m0.009s
user    0m0.000s
sys     0m0.006s
```
